# Expense Analytics Dashboard

## Overview
Interactive Power BI dashboard analyzing corporate expense reporting workflows, spending patterns, and approval bottlenecks across departments.

## Business Problem
Companies struggle to track expense inefficiencies — overspending departments, slow approval cycles, and category trends. This dashboard provides visibility into those issues.

## Dataset
Mock dataset simulating 300 corporate expense records across 6 departments and 7 expense categories (2024).

## Key Insights
- Operations and Marketing are the highest-spending departments
- Travel accounts for the largest expense category (~49% of total spend)
- Finance department has the longest average approval time
- Spending peaks in July and August

## Tools Used
- Power BI Desktop
- Microsoft Excel
- Power Query (data cleaning)

## Dashboard Features
- Spend by department (bar chart)
- Spend by category (donut chart)
- Monthly spend trend (line chart)
- Approval time by department (bar chart)
